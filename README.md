# Telegram-бот «Тотемное животное Московского зоопарка»

Telegram-бот для прохождения интерактивной викторины, которая определяет тотемное животное пользователя среди обитателей Московского зоопарка.

После прохождения теста пользователь получает:

- результат викторины;
- описание своего тотемного животного;
- фотографию животного;
- информацию о программе «Возьми животное под опеку»;
- возможность оставить отзыв;
- возможность связаться с представителями зоопарка.

---

## Возможности

### Викторина

- пошаговое прохождение теста;
- подсчёт баллов по выбранным ответам;
- определение наиболее подходящего животного;
- отображение результата с изображением и описанием.

### Работа с пользователем

- приветственное сообщение;
- встроенная справка;
- политика конфиденциальности;
- повторное прохождение теста.

### Дополнительные функции

- отзывы пользователей;
- сохранение отзывов в лог-файл;
- отправка сообщений представителям зоопарка;
- ссылка на программу опеки животных.

## Установка

### 1. Клонировать репозиторий

```bash
git clone <repository_url>
cd tgbot_py
```

### 2. Создать виртуальное окружение

Windows:

```bash
python -m venv venv
venv\Scripts\activate
```

Linux / macOS:

```bash
python3 -m venv venv
source venv/bin/activate
```

### 3. Установить зависимости

```bash
pip install -r requirements.txt
```

---

## Настройка

Создайте файл `.env` в корневой папке проекта.

Пример содержимого:

```env
BOT_TOKEN=your_bot_token
BOT_USERNAME=your_bot_username
ADMIN_CHAT_ID=123456789
CONTACT_EMAIL=contact@example.com
FEEDBACK_FILE=feedback.log
```

### Описание переменных

| Переменная | Описание |
|------------|-----------|
| BOT_TOKEN | Токен Telegram-бота от BotFather |
| BOT_USERNAME | Username бота без символа @ |
| ADMIN_CHAT_ID | Telegram ID администратора |
| CONTACT_EMAIL | Контактный email |
| FEEDBACK_FILE | Файл для хранения отзывов |

## Логика викторины

Пользователь отвечает на серию вопросов.

Каждый вариант ответа начисляет баллы одному или нескольким животным.

После завершения викторины система:

1. суммирует баллы;
2. определяет животное-победителя;
3. показывает результат;
4. отображает фотографию и описание животного.

---

## Программа опеки животных

После получения результата бот предлагает узнать больше о программе:

**«Возьми животное под опеку»**

Пользователь может перейти по ссылке и ознакомиться с условиями участия в программе поддержки животных Московского зоопарка.

---

## Отзывы

Бот позволяет пользователям:

- поставить оценку;
- оставить текстовый комментарий.

Отзывы сохраняются в файл:

```text
feedback.log
```

Формат записи:

```text
[2025-01-01 12:00:00] userId=123456 username=user rating=5 comment=Отличный бот!
```

## Конфиденциальность

Бот использует только данные, необходимые для:

- прохождения викторины;
- отправки обратной связи;
- связи с администрацией.

Персональные данные не передаются третьим лицам.
